MUZIKAZ HIGH-LEVEL IMAGE PACK

This package contains web-ready PNG crops, high-resolution generated artwork, editable SVG brand icons, and transparent-background attempts.

Suggested website use:
- Use 01_brand for header/footer logos.
- Use 02_hero/hero_banner_full.png for the homepage hero.
- Use 03_characters for trait overview content.
- Use 04_collection_tiles and 05_merch for product/collection cards.
- Use 06_ui_icons_buttons for buttons and navigation graphics.
- Use 09_section_banners for promotion rows.
- Use 12_transparent_attempts when you need no-background PNGs.

Important:
The supplied reference was a flattened JPG, so exact clean transparency is not possible without source layers. The transparent versions are automated dark-background removals and should be checked before final production.
